# MAIF Testing Framework

A plug-and-play backtesting framework for the Minutemen Alternative Investment Fund. Write a strategy, get a standardized scorecard — same metrics, same pipeline, comparable results.

**Site:** [maif.robbiew.dev](https://maif.robbiew.dev) &nbsp;·&nbsp; **Try it in Colab:** [quickstart notebook ↗](https://colab.research.google.com/github/robbiebusinessacc/maif-backtester/blob/main/notebooks/quickstart.ipynb)

## Quickstart (30 seconds)

```bash
pip install -r requirements.txt
python3 quickstart.py
```

This runs a sample strategy on SPY and produces a 4-page scorecard PNG. To test your own strategy, edit `quickstart.py` and replace `MyStrategy` with your logic.

## Writing a Strategy

Two modes — pick one:

### Signal mode (simple)

Return BUY/SELL/HOLD for each bar. The framework handles position sizing.

```python
from strategy.base import Strategy, Signal

class MyStrategy(Strategy):
    @property
    def name(self):
        return "My Strategy"

    def generate_signals(self, df):
        signals = pd.Series(Signal.HOLD, index=df.index)
        # Your logic here — set signals to Signal.BUY or Signal.SELL
        return signals
```

### Target position mode (advanced)

Specify exact position sizes. Supports long, short, and partial positions.

```python
class MyStrategy(Strategy):
    @property
    def name(self):
        return "My Strategy"

    def prepare(self, bars):
        df = bars.copy()
        df["signal_col"] = ...  # add indicators
        return df

    def target_position(self, row, state):
        # state.current_position, state.cash, state.equity available
        # Return positive = long, negative = short, 0 = flat
        return 100 if row["signal_col"] > 0 else 0
```

## Generating a Scorecard

```python
from backtester import BacktestConfig, generate_scorecard, EventDrivenBacktester

config = BacktestConfig(initial_capital=100_000)
ed = EventDrivenBacktester(config).run(strategy, df)

generate_scorecard(
    strategy=strategy,
    df=df,
    config=config,
    output_path="my_report.png",
    event_driven_results={"result": ed},
    theme="dark",  # or "light"
)
```

This produces 4 pages:
- **Page 1** — Equity curve, key metrics, trade stats
- **Page 2** — Monte Carlo stress tests (GBM noise, GAN regimes, noise injection)
- **Page 3** — Bar vs event-driven engine comparison
- **Page 4** — Letter grades across all dimensions

## Data Sources

```python

dl = DataLayer()
```

**Equities:** Yahoo Finance (free, no key), plus Alpaca, Polygon, Tiingo, etc.
**Crypto:** `CCXTProvider("kraken")` — 100+ exchanges, no API key for public OHLCV.
**Options:** `OptionsBacktester` with Black-Scholes pricing from underlying data.

### Crypto example

```python
from data_layer import CCXTProvider
```

## GAN Synthetic Data

Stress-test strategies against regime-conditioned synthetic paths generated by a trained cGAN.

```python
from backtester import make_gan_source, run_scenario_suite

crash_source = make_gan_source("crash", n_bars=252)
batch = run_scenario_suite(strategy, crash_source, n_scenarios=100)
print(batch.summary_frame)
```

Available regimes: `"bullish"`, `"bearish"`, `"sideways"`, `"crash"`, `"mixed"`.

## Built-in Strategies

| Strategy | Type | Description |
|---|---|---|
| `SMACrossover(fast, slow)` | Signal | Dual moving average crossover |
| `MeanReversion(lookback, entry_z)` | Target position | Z-score mean reversion with optional shorts |
| `BracketBreakout(lookback)` | Event-driven | Donchian breakout with OCO stop-loss/take-profit |
| `TurtleTrend(entry, exit)` | Target position | Turtle-style trend follower with ATR sizing |
| `CoveredCallStrategy(dte, delta)` | Options | Sell monthly covered calls on underlying |
